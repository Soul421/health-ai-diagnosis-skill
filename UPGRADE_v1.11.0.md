## 🎯 升级目标

从"方向性建议"升级为"拿起来就能用的方案"——每个场景都补充具体的工具组合、预算明细、落地步骤，让企业拿到报告就能直接选型、直接报价、直接启动。

## 📋 升级内容清单

### 1. 本体知识库升级（ontology.py）⭐ 核心升级

**10大场景全部工具化**，每个场景新增3件套：

| 新增字段 | 内容说明 |
|---------|---------|
| `tools` | 工具组合：主选方案（推荐工具+理由）+ 备选方案（2-3个替代产品） |
| `budget` | 预算明细：三档（轻量/标准/进阶），拆到工具费/实施费/培训费/运维费 |
| `timeline` | 落地步骤：按周拆解，每周任务+交付物 |

**场景覆盖**（10个全部完成）：
- ✅ 智能客服/咨询（智齿科技为主选）

- ✅ 门店服务监控（飞书录音豆为主选）⭐ 保持原有完整方案，作为质量标杆

### 2. 场景计算引擎升级（calculate_scenarios.py）

**新增输出字段**：
- `tools`：工具组合详情（主选+备选）
- `budget`：三档预算明细
- `timeline`：按周落地步骤
- `recommended_budget_tier`：根据企业规模自动推荐档位（light/standard/advanced）
- `budget_range`：推荐档位的预算摘要（首年/次年，万元）
- `implementation_weeks`：落地周期（周数）
- `recommended_tier`：整体推荐档位
- `top3_budget_estimate`：Top3场景总预算估算

**智能档位推荐逻辑**：
- 预算≥50万 → 进阶版
- 预算10-50万 → 标准版
- 预算<10万 → 轻量版
- 营收≥1亿 → 进阶版
- 营收1000万-1亿 → 标准版
- 营收<1000万 → 轻量版
- 员工≥200人 → 进阶版
- 员工50-200人 → 标准版
- 员工<50人 → 轻量版

## 💰 预算数据说明

所有工具和价格均基于2026年最新市场调研：

**数据来源**：
- 各厂商官方公开报价
- 行业评测报告（IT168、什么值得买、CSDN等）
- 第三方选型测评数据
- 实际成交价格区间

**价格浮动说明**：
- 表中价格为市场参考价，实际成交价受企业规模、定制需求、谈判力度影响，可能有20-30%浮动
- SaaS产品通常有年付优惠（10-20%）
- 私有化部署实施费通常为软件费的30-100%
