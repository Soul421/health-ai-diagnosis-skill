# AI应用场景优先级基准表（健康行业）

> 本文件为场景优先级动态计算的基准参考值。实际应用时需根据企业问卷回答动态调整各项参数，最终得分 = 痛点匹配度 × 技术成熟度 × ROI预期 ÷ 落地难度。

---

## 一、计算公式

```
场景综合得分 = 痛点匹配度 × 技术成熟度 × ROI预期 ÷ 落地难度
```

| 维度 | 评分范围 | 说明 |
|------|---------|------|
| 痛点匹配度 | 1-5分 | 该场景能多大程度解决企业Top痛点 |
| 技术成熟度 | 1-5分 | 技术是否成熟可靠，市场是否有成熟方案 |
| ROI预期 | 1-5分 | 投入产出比，越高越好 |
| 落地难度 | 1-5分 | 实施难度，越难得分越低（除数） |

> 注意：落地难度是除数，难度越高得分越低。

## 三、动态调整规则

### 3.1 痛点匹配度动态调整

根据问卷第16题（核心痛点选择）动态调整：

| 企业痛点 | 受影响场景 | 调整方式 |
|---------|-----------|---------|
| 客服人力成本高/响应不及时 | 智能客服 | 痛点匹配度+1 |
| 缺乏内容创作能力 | AI内容生成 | 痛点匹配度+1 |
| 客户流失/复购率低 | 私域运营AI | 痛点匹配度+1 |
| 数据分散/难以决策 | 数据分析AI | 痛点匹配度+1 |
| 团队管理难/人效低 | AI培训助手 | 痛点匹配度+1 |
| 获客成本高 | AI内容生成 + 私域运营AI | 各+0.5 |
| 内部流程繁琐/效率低 | 数据分析AI + 智能客服 | 各+0.5 |
| 门店服务质量差/合规风险 | 门店服务诊断 | 痛点匹配度+2 |

### 3.2 落地难度动态调整

根据问卷第11、18、20题（系统使用、AI认知、技术团队）动态调整：

| 企业条件 | 调整方式 |
|---------|---------|
| 无任何数字化系统 | 所有场景落地难度+1 |
| 有CRM/会员系统 | 私域运营AI 落地难度-1 |
| 有ERP/数据分析系统 | 数据分析AI 落地难度-1 |
| 无技术团队 | 定制类场景落地难度+1 |
| 有开发团队 | 所有场景落地难度-0.5 |
| 从未用过AI工具 | 所有场景落地难度+0.5 |
| 公司有AI应用经验 | 所有场景落地难度-1 |

### 3.3 ROI预期动态调整

根据问卷第7、9、19题（团队规模、营收、预算）动态调整：

| 企业条件 | 调整方式 |
|---------|---------|
| 客服团队≥10人 | 智能客服 ROI+1 |
| 内容/运营团队≥5人 | AI内容生成 ROI+1 |
| 门店≥10家 | 门店服务诊断 ROI+1 |
| 年营收≥5000万 | 所有场景ROI+0.5 |
| 预算＜5万 | 高成本场景ROI-1 |
| 预算≥50万 | 所有场景ROI+0.5 |

## 四、场景推荐输出规则

1. 计算所有AI场景的动态综合得分
2. 按得分从高到低排序
3. 报告中展示 Top 3 场景
4. 首选场景必须是得分最高的场景
5. 若门店诊断触发（检测到门店/连锁关键词），门店服务诊断场景自动进入Top 3
