# 短视频爆款结构库

> 版本：v2.0.0
> 适用：健康行业短视频（抖音/快手/视频号）
> 收录：8大类型 × 100+结构 × 适用场景

---

## 一、爆款视频的底层逻辑

### 健康行业爆款公式
```
爆款 = 精准痛点钩子 + 情绪共鸣 + 解决方案 + 信任背书 + 行动指令
```

### 3秒留人法则
- 第1秒：视觉冲击（人/物/场景）
- 第2秒：抛出问题/痛点
- 第3秒：给出承诺/反转

### 健康行业爆款5要素
1. **痛点要准**——精准戳中目标人群的核心焦虑
2. **切口要小**——一个视频只讲一个点
3. **反差要大**——认知反差/前后反差/预期反差
4. **信任要足**——资质/案例/数据/专家
5. **行动要强**——明确告诉观众做什么

## 二、8大爆款结构模板

### 结构1：痛点+解决方案型（最常用）

**适用：** 所有产品、所有平台
**成功率：** ★★★★★
**平均时长：** 30-60秒

**结构拆解：**
```
0-3s  钩子（痛点提问）
3-10s 共鸣（你是不是也...）
10-20s 原因（为什么会这样）
20-40s 方案（怎么办+产品）
40-50s 证据（案例/数据）
50-55s 行动（关注/直播间/评论区）
```

**示例脚本（羊奶粉）：**
| 时间 | 画面 | 台词 | 字幕 |
|------|------|------|------|
| 0-3s | 阿姨捂肚子痛苦表情 | "肠胃不好的叔叔阿姨，先别划走！" | 肠胃不好的看过来 |
| 3-10s | 多个不适场景混剪 | "是不是经常便秘、腹胀、喝牛奶还拉肚子？吃了好多东西都不吸收？" | 便秘？腹胀？不吸收？ |
| 10-20s | 科普图解 | "其实啊，不是你的问题，是牛奶里的乳糖不耐受。很多中老年人都有这个情况。" | 乳糖不耐受 很多人都有 |
| 20-40s | 产品展示+冲泡 | "那怎么办呢？试试羊奶粉。羊奶的分子更小，更容易吸收，而且乳糖含量低，很多喝牛奶不舒服的人，喝羊奶基本都没事。" | 羊奶粉 好吸收 |
| 40-50s | 用户反馈截图 | "你看张阿姨喝了一个月，现在胃口都好多了。" | 真实反馈 |
| 50-55s | 主播对着镜头 | "想试试的，点我头像来直播间，今天有活动价。" | 来直播间有活动 |

**变体：**
- 痛点可以换成"误区"（别再XX了）
- 方案可以换成"妙招"（教你一招）
- 证据可以换成"对比"（以前VS现在）

---

### 结构2：对比反转型

**适用：** 有明显差异化的产品
**成功率：** ★★★★☆
**平均时长：** 20-40秒

**结构拆解：**
```
0-3s  错误做法（常见误区）
3-10s  为什么错（危害/原因）
10-20s 正确做法（产品）
20-30s 对比效果（好在哪里）
30-35s 行动引导
```

**示例脚本（益生菌）：**
| 时间 | 画面 | 台词 |
|------|------|------|
| 0-3s | 普通酸奶特写 | "你还在靠喝酸奶补益生菌？" |
| 3-10s | 摇头+数据图 | "大错特错！酸奶里的益生菌，经过胃酸早就死得差不多了，到肠道里没剩几个。" |
| 10-20s | 益生菌产品特写 | "要补益生菌，得选这种活性益生菌粉，每袋有几百亿活菌，还有包埋技术，活着到肠道。" |
| 20-30s | 对比表格 | "你看，一袋顶得上几十瓶酸奶的量，价格还便宜。" |
| 30-35s | 小黄车指一下 | "想要的，小黄车直接拍。" |

---

### 结构3：故事案例型

**适用：** 建立信任、高客单价产品
**成功率：** ★★★★☆
**平均时长：** 60-90秒

**结构拆解：**
```
0-5s   人物出场（他是谁+什么问题）
5-20s  遇到的困难（有多惨）
20-30s 转折点（怎么遇到产品的）
30-50s 使用过程（怎么用的）
50-70s 现在的变化（效果）
70-85s 总结+行动
```

**关键要点：**
1. 人物要真实（有细节
2. 困难要具体（有画面感）
3. 转折要自然（不生硬）
4. 变化要明显（有对比）

> ⚠️ 合规提醒：故事案例不能涉及疾病治疗，只能说"调理/改善/舒服了/精神好了"，不能说"治好/痊愈/病好了"

---

### 结构4：科普知识型

**适用：** 专业人设、信任建立
**成功率：** ★★★☆☆
**平均时长：** 40-60秒

**结构拆解：**
```
0-3s  问题（你知道XX吗？）
3-10s 误区（90%的人都错了）
10-30s 讲解（到底是怎么回事）
30-45s 建议（应该怎么做）
45-55s 产品植入（推荐XX）
55-60s 关注引导
```

**选题方向：**
- "XX的3个信号"
- "XX到底好不好？"
- "一文讲清XX"
- "你被骗了多少年？"
- "医生/营养师怎么看XX"

---

### 结构5：测评体验型

**适用：** 新产品、竞品多的品类
**成功率：** ★★★★☆
**平均时长：** 60-90秒

**结构拆解：**
```
0-5s   引入（今天测了X款产品）
5-20s  第1款（优点+缺点）
20-35s 第2款（优点+缺点）
35-50s 第3款（优点+缺点）
50-70s 对比总结（哪款值得买）
70-85s 推荐理由+购买引导
```

**测评维度：
- 成分/配方
- 口感/味道
- 效果/感受
- 价格/性价比
- 品牌/资质

> ⚠️ 合规提醒：测评要客观，不能恶意贬低竞品，对比要有依据

---

### 结构6：避坑提醒型

**适用：** 引流爆款率最高的类型之一
**成功率：** ★★★★★
**平均时长：** 30-50秒

**结构拆解：**
```
0-3s  警告（千万别再XX了！）
3-10s  为什么（有什么危害/坑）
10-25s 常见的几个坑（1...2...3...）
25-40s 正确做法（应该怎么选）
40-50s 推荐产品+行动
```

**爆款选题方向：**
- "买XX的3个坑，第3个最容易中招"
- "千万别买错XX，90%的人都踩坑了"
- "XX的真相，商家不会告诉你"
- "这种XX别再吃了，我也是刚知道"

---

### 结构7：好物推荐型

**适用：** 直接转化、直播引流
**成功率：** ★★★☆☆
**平均时长：** 20-30秒

**结构拆解：**
```
0-3s  产品亮相
3-10s 为什么好（1-2个核心卖点）
10-20s 价格/优惠（今天XX钱）
20-25s 行动引导
```

**适用场景：**
- 粉丝已经有信任基础的账号
- 直播切片
- 直播间引流
- 复购客户

---

### 结构8：日常VLOG型

**适用：** 人设打造、情感连接
**成功率：** ★★★☆☆
**平均时长：** 40-60秒

**结构拆解：**
```
0-5s   日常场景（开店/上班/见客户）
5-20s  发生了什么事
20-40s  自然植入产品
40-55s  感悟/总结
55-60s  关注引导
```

**作用：**
- 建立人设真实感
- 拉近距离
- 增加信任
- 提升粘性

## 六、AI辅助创作指南

### 选题生成Prompt
```
帮我生成20条[产品名]的短视频选题，目标人群是[中老年人/宝妈/上班族]，
类型要多样化，包含痛点型、科普型、避坑型等，爆款指数要高
```

### 脚本生成Prompt
```
把以下产品卖点改写成30秒短视频脚本，用痛点+解决方案结构，
开头要有钩子，结尾要有行动引导，产品是[产品名]，卖点是[卖点1、2、3]
```

### 标题生成Prompt
```
给这条视频生成10个标题，要符合抖音风格，3秒抓眼球，
视频内容是[内容简介]
```

### 字幕优化Prompt
```
把这段台词改成短视频字幕，每屏不超过15字，要有节奏感，
重点词加粗，突出痛点和利益点
```
