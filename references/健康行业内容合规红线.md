# 健康行业内容合规红线手册

> 版本：v2.0.0
> 适用：健康食品、保健品、医疗器械、养生服务等健康行业内容创作
> 更新日期：2025年

---

## 一、核心法规依据

| 法规名称 | 适用范围 | 重点关注 |
|----------|---------|---------|
| 《广告法》 | 所有广告宣传 | 绝对化用语、虚假宣传 |
| 《食品安全法》 | 食品、保健食品 | 功能宣称、标签标识 |
| 《保健食品广告审查暂行规定》 | 保健食品广告 | 广告审查、声称功能 |
| 《药品广告审查发布标准》 | 药品广告 | 药品广告发布规范 |
| 《医疗器械广告审查发布标准》 | 医疗器械广告 | 医疗器械广告规范 |
| 《互联网广告管理办法》 | 网络广告 | 互联网广告合规 |

## 二、绝对不能碰的10条红线

### ❌ 红线1：涉及疾病治疗功能
**禁止表述：**
- 治疗、治愈、疗效、痊愈、医治
- 根治、治好、药到病除
- 防病、治病、诊病、就医
- 中西医结合治疗、临床验证
- 治愈率、有效率、康复率

**合规替换：**
- "有助于改善XX" → 代替 "治疗XX"
- "对XX有帮助" → 代替 "治好了XX"
- "调理XX" → 代替 "治疗XX病"
- "养护XX" → 代替 "医治XX"

> ⚠️ 注意：普通食品不能宣称任何保健功能；保健食品只能宣称批准的27种保健功能。

### ❌ 红线2：绝对化用语
**禁止词汇：**
- 最：最好、最佳、最有效、最便宜、最新技术
- 第一：全国第一、行业第一、销量第一
- 级：顶级、极品、世界级、国家级
- 绝：绝对、绝无仅有、绝对有效
- 永久：永久有效、永不复发、终身受益

**合规表达：**
- "深受XX人群选择" → 代替 "销量第一"
- "很多用户反馈不错" → 代替 "效果最好"
- "我们的核心优势是" → 代替 "顶级"

### ❌ 红线3：虚假承诺效果
**禁止承诺：**
- 保证效果、无效退款（需符合消法，但不可夸大）
- 100%有效、人人都能用
- 用了就好、包你满意
- 永不反弹、彻底根治

**合规做法：**
- 效果表述为"因人而异"
- 展示真实用户反馈时标注"个体差异，效果不代表所有人"

### ❌ 红线4：利用患者/专家名义证明
**禁止行为：**
- 使用患者形象做前后对比
- 患者现身说法推荐产品
- 医生/专家穿白大褂推荐
- 引用某某医院/医生的推荐
- "某某医生都在用"

**合规做法：**
- 可以做科普，但不能推荐具体产品
- 用户案例需获得授权，且不能涉及疾病治疗
- 专家身份需真实可验证

### ❌ 红线5：与药品对比
**禁止表述：**
- 比吃药还管用
- 替代药物
- 是药三分毒，不如用我们的
- 比XX药效果还好
- 可以停药

**合规做法：**
- 不和药品做任何功效对比
- 保健食品标注"本品不能代替药物"
- 不暗示可以替代药物治疗

### ❌ 红线6：宣称保健食品超范围宣传
**保健食品允许声称的27种保健功能：**
1. 增强免疫力
2. 辅助降血脂
3. 辅助降血糖
4. 抗氧化
5. 辅助改善记忆
6. 缓解视疲劳
7. 促进排铅
8. 清咽
9. 辅助降血压
10. 改善睡眠
11. 促进泌乳
12. 缓解体力疲劳
13. 提高缺氧耐受力
14. 对辐射危害有辅助保护功能
15. 减肥
16. 改善生长发育
17. 增加骨密度
18. 改善营养性贫血
19. 对化学性肝损伤有辅助保护作用
20. 祛痤疮
21. 祛黄褐斑
22. 改善皮肤水分
23. 改善皮肤油分
24. 调节肠道菌群
25. 促进消化
26. 通便
27. 对胃粘膜有辅助保护功能

> ⚠️ 注意：只能宣传已批准的功能，且必须加"辅助"二字（如"辅助降血糖"，不能说"降血糖"）

### ❌ 红线7：虚假资质/背书
**禁止行为：**
- 伪造检测报告、认证证书
- 冒充知名品牌合作
- 虚构权威机构认证
- P图修改资质文件
- 谎称"央视上榜品牌"等无法验证的荣誉

### ❌ 红线8：价格欺诈
**禁止行为：**
- 虚构原价、虚假打折
- "全网最低价"（没有依据）
- 虚假限购（实际不限量）
- 假秒杀（长期挂着秒杀价）
- 价格对比无依据

**合规做法：**
- 原价要有真实销售记录
- 价格对比要有明确对比依据
- 限时限量要真实

### ❌ 红线9：贬低竞品
**禁止行为：**
- 直接点名竞品不好
- 暗示竞品有问题
- "别人家的都是假的，只有我们是真的"
- 恶意对比、踩一捧一

**合规做法：**
- 只说自己的好，不说别人的坏
- 对比要客观，有数据支撑
- 不点名具体竞品

### ❌ 红线10：诱导分享/强制拉人
**禁止行为：**
- "转发朋友圈才能领"（诱导分享）
- "拉3个人进群送礼品"（裂变拉新）
- 强制关注、强制分享
- 虚假抽奖、虚假福利

## 三、不同内容形式的合规要点

### 3.1 短视频合规

| 检查项 | 要求 |
|--------|------|
| 视频口播 | 不能有违禁词 |
| 字幕文字 | 不能有违禁词 |
| 画面展示 | 不能有前后对比（疾病类） |
| 评论区 | 设置关键词过滤，及时删除违规评论 |
| 购物车标题 | 商品名称要规范，不能夸大 |
| 背景音乐 | 不能用医疗相关背景 |
| 人物着装 | 不能穿白大褂卖普通食品 |

### 3.2 直播合规

| 检查项 | 要求 |
|--------|------|
| 主播话术 | 不能有违禁词 |
| 产品介绍 | 按批准内容宣传 |
| 效果展示 | 不能展示疾病治疗效果 |
| 价格宣传 | 真实有据 |
| 赠品宣传 | 如实说明 |
| 售后承诺 | 不夸大承诺 |
| 保健食品 | 必须口播"本品不能代替药物"（每场至少3次） |

### 3.3 朋友圈/社群合规

| 检查项 | 要求 |
|--------|------|
| 文案内容 | 不能有违禁词 |
| 用户反馈 | 获得授权，不涉及治疗 |
| 转账截图 | 不能造假 |
| 活动宣传 | 真实有效 |
| 专家背书 | 真实可验证 |

### 3.4 产品详情页合规

| 检查项 | 要求 |
|--------|------|
| 标题 | 不夸大 |
| 主图 | 不使用医疗术语 |
| 详情页 | 按批准内容 |
| 资质展示 | 真实有效 |
| 用户评价 | 不刷好评 |
| 功效宣传 | 不超范围 |

## 四、常见违规词替换表

| 违规词 | 合规替换词 |
|--------|-----------|
| 治疗、治愈 | 调理、改善、养护、有助于 |
| 根治、彻底 | 深入、深度、有效改善 |
| 最、第一、顶级 | 很、非常、深受欢迎、很多人选择 |
| 100%有效、保证效果 | 很多用户反馈不错、效果因人而异 |
| 永不反弹、不复发 | 持续使用效果更佳 |
| 药到病除、立竿见影 | 循序渐进、坚持使用有帮助 |
| 防癌、抗癌 | （绝对不能说，删除） |
| 比吃药好、替代药物 | （绝对不能说，删除） |
| 国家级、世界级 | 行业内、市场上 |
| 无效退款 | 7天无理由退换（符合消法的表述） |
| 纯天然、无任何副作用 | 成分天然提取、温和不添加XX |
| 祖传秘方、独家配方 | 传统配方、经典配方 |

## 五、平台规则要点

### 抖音/快手
- 保健品类目需要报白
- 医疗器械需要资质
- 违禁词检测严格，AI+人工双重审核
- 违规处罚：限流、下架、封号、罚款

### 视频号
- 微信生态内合规要求更严
- 诱导分享处罚重
- 链接跳转受限

### 小红书
- 种草笔记不能太硬广
- 需要标注"赞助"或"合作"
- 医疗健康内容审核严格

### 淘宝/京东
- 商品详情页合规要求高
- 有专门的合规审核
- 投诉举报处理严格

## 六、合规审核流程（建议）

```
内容创作 → AI初检（违禁词） → 人工审核（专业合规）
   → 修改调整 → 确认无误 → 发布上线
   → 持续监控（评论区/弹幕/私信） → 问题及时处理
```

### 审核清单（每条内容发布前必查）

- [ ] 没有涉及疾病治疗功能的表述
- [ ] 没有使用绝对化用语
- [ ] 保健食品已标注"本品不能代替药物"（如适用）
- [ ] 没有使用患者/医生名义推荐
- [ ] 没有与药品对比
- [ ] 功效宣传在批准范围内
- [ ] 价格宣传真实有据
- [ ] 资质文件真实有效
- [ ] 没有贬低竞品
- [ ] 用户案例已获得授权

## 七、违规后果与应对

### 违规后果
1. **轻度**：限流、内容下架
2. **中度**：账号降权、短期封禁
3. **重度**：账号永久封禁
4. **严重**：市场监管部门处罚、罚款
5. **特别严重**：刑事责任

### 应对建议
1. 发现违规立即删除/下架
2. 不要与平台对抗，申诉要有理有据
3. 保留所有资质文件备查
4. 定期培训团队合规意识
5. 建议聘请专业法律顾问
