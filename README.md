# 健康行业AI落地快速诊断 Skill

## 一句话介绍

帮健康行业企业家快速判断AI是否适合自己、怎么落地、怎么不踩坑

---

## 适用场景

- 健康行业企业家想了解AI能否帮到自己
- 企业数字化转型负责人需要AI落地规划
- 咨询顾问需要快速诊断客户AI准备度
- 投资人评估企业AI应用潜力

---

## 目标用户

- 健康行业企业家
- 数字化转型负责人
- 企业运营总监/CEO
- 行业咨询顾问

---

## 核心价值

1. **专业判断**：把"AI适不适合我"的判断变成可执行的诊断流程
2. **10分钟出报告**：快速诊断，输出可落地的建议
3. **案例沉淀**：积累行业案例，反哺公众号内容
4. **专业服务入口**：诊断报告作为深度合作的起点，提供后续专业支持

---

## 使用方法

### Step 1：让用户填写诊断问卷

发送 `templates/诊断问卷.md` 给用户填写

### Step 2：收集问卷并计算得分

根据问卷答案，计算各维度得分：
- 数字化基础分：/25
- 经营规模分：/20
- 痛点紧迫度分：/25
- 资源准备度分：/15
- 团队接受度分：/15

### Step 3：生成诊断报告

基于得分和问卷内容，填充 `templates/报告模板-完整版.md` 或 `templates/报告模板-简化版.md`

### Step 4：专家审核与调整

诊断师审核报告合理性，补充行业洞察

### Step 5：交付与跟进

发送报告给客户，预约解读会议
