# AI知识库与团队执行模板

> 适用：健康行业内容团队、销售团队、运营团队
> 目标：让AI成为团队的"超级员工"，30天落地执行

---

## 一、AI知识库架构

### 知识库分层设计

| 层级 | 知识库名称 | 内容 | 使用人群 | 更新频率 |
|------|-----------|------|---------|---------|
| L1 | 产品知识库 | 产品知识、FABE、FAQ | 全员 | 每月更新 |
| L2 | 行业知识库 | 行业知识、科普内容、竞品信息 | 内容+销售 | 每季度更新 |
| L3 | 销售话术库 | 话术模板、异议处理、案例库 | 销售+主播 | 持续更新 |
| L4 | 内容素材库 | 选题、脚本、文案、图片 | 内容团队 | 每周更新 |
| L5 | 客户案例库 | 用户故事、反馈、前后对比 | 销售+内容 | 持续更新 |

### 知识库建设路径

```
第1步：资料收集 → 第2步：结构化整理 → 第3步：导入AI工具
   → 第4步：训练调优 → 第5步：团队使用 → 第6步：持续迭代
```

### AI内容生产流程设计

#### 短视频生产流水线（人+AI）

```
选题策划（人定方向）
    ↓
AI生成脚本初稿（AI）
    ↓
人工审核修改（人·10分钟）
    ↓
AI生成图片/数字人（AI）
    ↓
AI剪辑+字幕（AI）
    ↓
人工审核发布（人·5分钟）
```

**效率提升：** 从每条2小时 → 每条20分钟，效率提升6倍

#### 私域内容生产流水线

```
每周内容规划（人定主题）
    ↓
AI生成7天朋友圈文案（AI）
    ↓
AI生成社群每日内容（AI）
    ↓
人工审核调整（人·30分钟/周）
    ↓
定时发布（工具自动）
```

**效率提升：** 从每周8小时 → 每周1小时，效率提升8倍

## 六、AI使用规范

### 内容生产Prompt模板库

| 用途 | Prompt模板 |
|------|-----------|
| 短视频选题 | "帮我生成20条【产品名】的短视频选题，目标人群是XX，类型要多样化" |
| 短视频脚本 | "把以下产品卖点改写成30秒短视频脚本，开头要有钩子" |
| 直播话术 | "帮我写一段【产品名】的直播逼单话术，用限时+赠品组合方式" |
| 朋友圈文案 | "帮我写5条【产品名】的朋友圈文案，要真实不浮夸，适合XX风格" |
| 社群话术 | "帮我设计一套新群7天SOP话术，目标人群是XX" |
| 异议处理 | "用户说【异议】，帮我写3种不同风格的回答" |
| 用户案例 | "把这个用户反馈改写成有感染力的案例故事，200字以内" |

### 合规审核流程

```
AI生成内容 → 人工合规审核 → 修改调整 → 发布使用
       ↑                                    ↓
   反馈优化 ←──── 数据复盘 ←──── 效果跟踪
```
